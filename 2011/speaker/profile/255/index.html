Speaker Profile

Sinker Li

Sinker Li comes from Taiwan. He is a freelancer providing technical consulting for embedded Linux/FreeBSD systems at Taiwan. He is also an open source developer for 10+ years. He once hold a small Python conference at Taiwan in 2008, and is preparing a small Python conference at Taiwan in May 2011.

In last two years, he focus on the development of an GUI environment based on SVG. The project include an authoring tool that extend Inkscape with Python code, embedding CPython engine. The tool is for creating GUI and animation for applications. It is supposed to be a tool for fast-prototyping applications, and will export CSS3/HTML compatible code.

Site built by Eldarion using Django and Pinax.
