Speaker Profile

Dave Malcolm

Dave Malcolm lives in Boston, MA, where he works for Red Hat, improving Python itself, and maintaining the Python stacks within the Fedora Project and the Red Hat Enterprise Linux product. He's the developer of various tools to make it easier to debug Python (including teaching the "gdb" debugger about the insides of the CPython runtime).
