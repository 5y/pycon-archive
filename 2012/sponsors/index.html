IPython is an open source project that provides tools for interactive and parallel computing. It is best known for its enhanced Python shell whose features streamline interactive work: tab completion, integrated help, GUI toolkit support, system shell access, numbered prompts, command history and others.

A powerful new architecture for remote code execution has opened the door for a number of new IPython components. A Qt based console provides an IPython shell with inline figures, graphical calltips and persistent sessions. A high-performance, low-latency parallel computing framework provides abstractions for parallelizing Python code on multicore CPUs, clusters, supercomputers and the cloud. A web based notebook provides a user interface and open document format that mixes code with rich text, figures, mathematical equations and arbitrary HTML. Today,

IPython is included in all Linux distributions, and in the Enthought and Python(x,y) Distributions. It is part of Microsoft's Visual Studio Tools for Python and the PyCharm IDE, and is a component of a multitude of scientific computing projects in many disciplines, from mathematical environments to hardware control in large experimental facilities.
